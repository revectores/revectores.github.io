Text Decoration Property

Three properties are included for text decoration: text-decoration sets the apparence of decorative line(for example, underline) of text, text-emphasis controls the emphasis mark to text, and text-shadow makes a shadow from the text.

  1. text-emphasis is a property to mark emphasis symbol to text, which is a shorthand for

    • text-emphasis-style
    • text-emphasis-color.

Refer to the document of text-emphasis or the blog from Xinxu Zhang for more details.

  1. text-decoration sets the apparence of decorative line, which is a shorthand for
  • text-decoration-line
    • text-decoration-color
    • text-decoration-style
    • text-decoration-thickness

Refer to the document of text-decoration for more details.

  1. text-shadow controls the style of shadow of text. Refer to the document of text-shadow for more details.
